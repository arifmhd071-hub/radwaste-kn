🔥 Setup Firebase

Firebase menyimpan data entri limbah secara realtime dan menangani autentikasi pengguna.

  • Buat project Firebase baru
    Buka console.firebase.google.com ↗
    Klik Add project → beri nama radwaste-kn → klik Continue sampai selesai.
  • Daftarkan Web App & salin config
    Di halaman project, klik ikon </> (Web) → beri nama app → Register.
    Salin blok firebaseConfig yang muncul.

    Buka file public/index.html, cari baris GANTI_DENGAN_API_KEY_ANDA dan tempel nilai dari Firebase:
    public/index.html — bagian firebaseConfig 
      const firebaseConfig = {
    apiKey:            "AIzaSy...",         // dari Firebase Console
    authDomain:        "radwaste-kn.firebaseapp.com",
    projectId:         "radwaste-kn",
    storageBucket:     "radwaste-kn.appspot.com",
    messagingSenderId: "123456789",
    appId:             "1:123:web:abc123"
  };
  • Aktifkan Firestore Database
    Sidebar Firebase → Firestore Database → Create database
    Pilih Production mode → pilih region asia-southeast1 (Singapore) → Enable.
  • Set Firestore Security Rules
    Firestore → tab Rules → tempel rules berikut → klik Publish:
    Firestore Rules 
    rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId}/entries/{entryId} {
      allow read, write: if request.auth != null
        && request.auth.uid == userId;
    }
  }
}
  • Aktifkan Authentication → Email/Password
    Sidebar → Authentication → Sign-in method → klik Email/Password → toggle Enable → Save.

🐙 Upload ke GitHub

Repo GitHub diperlukan agar Netlify bisa otomatis deploy setiap Anda push perubahan.

⚠ Pastikan Anda sudah mengisi firebaseConfig di index.html sebelum push.
  • Buat repo baru di GitHub
    Buka github.com/new ↗
    Nama repo: radwaste-kn — pilih Public — klik Create repository.
    Jangan centang "Initialize with README".
  • Init Git & push dari terminal lokal
    Ekstrak folder radwaste-kn yang telah diunduh, lalu jalankan:
    Terminal 
    # Masuk ke folder proyek
cd radwaste-kn

# Inisialisasi git
git init
git add .
git commit -m "feat: RadWaste KN PWA with Firebase"

# Hubungkan ke GitHub (ganti USERNAME)
git remote add origin https://github.com/USERNAME/radwaste-kn.git
git branch -M main
git push -u origin main
  • Verifikasi upload berhasil
    Buka github.com/USERNAME/radwaste-kn — pastikan terlihat folder public/, file netlify.toml, dan README.md.

🌐 Deploy ke Netlify

Netlify meng-host aplikasi Anda dengan HTTPS otomatis — syarat wajib agar PWA bisa dikonversi ke APK.

  • Login ke Netlify
    app.netlify.com ↗ — login dengan akun GitHub Anda.
  • Import project dari GitHub
    Klik Add new site → Import an existing project → pilih GitHub.
    Authorize Netlify → pilih repo radwaste-kn.
  • Konfigurasi build (sudah otomatis via netlify.toml)
    Netlify akan mendeteksi netlify.toml secara otomatis.
    Pastikan Publish directory terisi public.
    Build command: kosongkan (tidak ada proses build).
  • Klik Deploy site & tunggu
    Deploy biasanya selesai dalam 30–60 detik.
    Setelah selesai, Anda mendapat URL seperti: https://radwaste-kn-abc123.netlify.app

    Opsional — ganti subdomain:
    Site settings → Domain management → Edit site name → ubah ke radwaste-kn.
  • Uji coba di browser
    Buka URL Netlify di browser → coba login/daftar → tambah entri.
    Buka DevTools → Application → Service Workers → pastikan SW activated.

⚠ Tambahkan Domain Netlify ke Firebase

Firebase Auth akan menolak login dari domain yang tidak terdaftar. Tambahkan domain Netlify Anda:

  • Firebase Console → Authentication → Settings → Authorized domains
    Klik Add domain → masukkan URL Netlify Anda (mis. radwaste-kn.netlify.app) → Save.

📦 Konversi ke APK via PWABuilder

PWABuilder mengemas PWA Anda menjadi file APK yang bisa diinstal di Android tanpa perlu Play Store.

  • Buka PWABuilder
    pwabuilder.com ↗
  • Masukkan URL Netlify Anda
    Ketik URL (mis. https://radwaste-kn.netlify.app) → klik Start.
    PWABuilder akan menganalisis Manifest, Service Worker, dan HTTPS.
    Semua indikator harus hijau ✓.
  • Klik "Package for stores"
    Setelah analisis selesai, klik tombol Package for stores di bagian bawah hasil.
  • Pilih Android → Generate
    Klik kartu Android → Generate.
    Isi form:
    • Package ID: com.namars.radwaste_kn
    • App name: RadWaste KN
    • App version: 1
    • App version string: 1.0.0
    • Semua field lain bisa dibiarkan default
  • Download package
    Klik Download → Anda mendapat file ZIP berisi:
    app-release-signed.apk — install langsung ke Android
    app-release.aab — upload ke Google Play Store (opsional)

🔍 Checklist sebelum generate APK

  • ☑ URL Netlify bisa diakses & HTTPS aktif
  • ☑ manifest.json valid (ada name, icons, start_url)
  • ☑ Service Worker terdaftar di DevTools → Application
  • ☑ Icon 512×512 tersedia (/icons/icon-512.png)
  • ☑ Sudah bisa login & simpan data di Netlify

📱 Install APK ke Android

Ada dua cara menginstal APK — via USB (ADB) atau langsung dari file manager HP.

  • Cara A — Transfer file & install manual
    1. Kirim file app-release-signed.apk ke HP (WhatsApp, email, atau Google Drive)
    2. Buka File Manager di HP → temukan file APK
    3. Tap file → Install
    4. Jika muncul peringatan "Install from unknown source" → buka Settings → Install unknown apps → aktifkan untuk app yang digunakan
    5. Kembali dan install ulang
  • Cara B — Via ADB (USB Debugging)
    Aktifkan USB Debugging di HP (Settings → Developer options), lalu:
    Terminal 
    # Pastikan HP terhubung via USB
adb devices

# Install APK
adb install app-release-signed.apk

# Atau jika replace versi lama
adb install -r app-release-signed.apk
  • Verifikasi instalasi
    Cari RadWaste KN di app drawer HP → tap untuk membuka.
    Login dengan akun yang sudah dibuat → data akan sinkron dari Firebase.
